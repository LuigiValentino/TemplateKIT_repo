TemplateKit Documentation

Modular framework for creating reusable HTML, CSS, and JavaScript components.

Introduction

TemplateKit is a modular framework that allows you to create reusable components in HTML, CSS, and JavaScript. The goal is to simplify the development of web pages by breaking them into smaller, manageable parts.

View on GitHub

Getting Started

To start using TemplateKit, the first step is to clone the repository from GitHub. Follow these steps:

  1. Step 1: Open your terminal and run the following command to clone the repository:
    2. git clone https://github.com/LuigiValentino/TemplateKIT_repo
  2. Step 2: Navigate into the cloned directory:
    3. cd templatekit
  3. Step 3: Start developing your project by using the modular components provided in the components folder.

Installation

To integrate TemplateKit into your project, follow these steps:

  1. Step 1: Include the templatekit.js script in your project:
    2. <script src="path/to/templatekit.js"></script>
  2. Step 2: Create a components directory in your project, where all your components will reside.

Usage

Once you've installed TemplateKit, you can start creating and using components in your HTML file. Here's an example:

Basic Usage

In your index.html file, use custom tags like <navbar-component> to load a Navbar component.

<navbar-component></navbar-component>

This custom tag will automatically load the navbar.html component into the page.

Creating Components

Components are the core of TemplateKit. Each component consists of three parts: HTML, CSS, and JavaScript. Here's how to create a simple component:

1. HTML

Create an HTML file for your component, for example, navbar.html:

<nav>
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>

2. CSS

Create a CSS file for styling your component, for example, navbar.css:

nav {
    background-color: #3498db;
    padding: 10px;
}

nav ul {
    list-style-type: none;
    text-align: center;
}

nav ul li {
    display: inline-block;
    margin-right: 10px;
}

3. JavaScript (Optional)

If your component needs interactivity, you can add JavaScript in a navbar.js file:

document.querySelectorAll('nav a').forEach(link => {
    link.addEventListener('click', () => {
        alert('Navbar link clicked!');
    });
});
